# pasteque

###### 0.0.8

Beautify typography.

**pasteque** is a fork from [**Compass typography**](https://github.com/Compass/compass/tree/stable/core/stylesheets/compass/typography).

### Installation

```
$ npm install pasteque
```

### Usage

```scss
@import "path/to/node_modules/pasteque/pasteque";
```
