# pasteque-unit

###### 0.1.0

Beautify typography:
* Unit.

**pasteque-unit** is a fork from [**Compass typography**](https://github.com/Compass/compass/tree/stable/core/stylesheets/compass/typography).

### Installation

```
$ npm install pasteque-unit
```

### Usage

```scss
@import "path/to/node_modules/pasteque-unit/unit";
```
