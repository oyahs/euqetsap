# pasteque-unit

###### 0.0.6

Beautify typography:
* Unit.

**pasteque-unit** is a fork from [**Compass**](https://github.com/Compass/compass).

### Installation

```
$ npm install pasteque-unit
```

### Usage

```scss

```
